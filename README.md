# 🧙‍♂️ Harry Potter Wiki

An interactive Harry Potter Wiki built using **React, Redux Toolkit, Redux Thunk, and React Query** to demonstrate advanced **state management and caching mechanisms**.

This project showcases how different data fetching and caching strategies can be implemented in modern React applications.

---

# 🚀 Features

- 📚 Books Page using **Redux + Thunk**
- ✨ Spells Page with **Pagination**
- 👤 Characters Page using **React Query**
- 🏰 Houses Page with **Query Invalidation**
- ⚡ Advanced caching using **React Query**
- 🎨 Clean UI and responsive design
- 🔄 API data fetching with optimized caching strategies
